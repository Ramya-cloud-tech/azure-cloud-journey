# Day 1 - What is Cloud Computing? ☁️

Date: 19 June 2026

## What is cloud computing?

Cloud computing means renting IT resources over the internet instead of buying and maintaining them yourself.

Examples:

* Servers
* Storage
* Databases
* Networking
* Security

## Why do companies use cloud?

Benefits:

* Lower cost
* Easier to manage
* Expand anytime
* Faster deployment

## Cloud Models

### Public Cloud

Resources owned by Microsoft and shared among customers.

### Private Cloud

Resources owned by one organization.

### Hybrid Cloud

Combination of public and private cloud.
